# Guru Ads

## Ver 1.0.0


## 简介

Guru 的广告框架库

用于实现各个广告聚合平台广告初始化，展示和事件回调的功能

当前版本支持了 AppLovin Max 聚合相关的广告渠道和对应的实现逻辑


## 支持的广告类型：

* Banner

    底部的 Banner 广告


* Interstitial

    关卡之间的插屏广告， 自动弹出，无奖励


* Rewarded

    可赋予用户奖励道具的激励视频
